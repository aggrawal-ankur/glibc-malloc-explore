# There are no data types, only bits, widths and interpretations.

Every programming language has data types.
  - Some are dynamically typed, where data types are not enforced strictly on the variables. Example: Python or JavaScript.
  - Other languages are statically typed, where types are "strictly enforced". Example: C.

C has a bucket full of types. Some of them are really popular popular, the others are popular in niche communities.

Here is a complete list of data types in C:

| Data Type | Format Specifier | Width On 32-bit | Width On 64-bit | Range of Values | Total Addressable Values (2^n_bits) | Signedness |
| :-------- | :--------------- | :-------------- | :-------------- | :-------------- | ----------------------------------: | :--------- |
| char      | %c | 1 byte | 1 byte |  Implementation-defined | 256 | Implementation-defined |
| unsigned char | %c | 1 byte | 1 byte | [0, (2^8)-1] | 256 | unsigned |
| signed char   | %c | 1 byte | 1 byte | [-(2^7), (2^7)-1] | 256 | signed |
| |
| unsigned short, unsigned short int | %hu | 2 byte | 2 byte | [0, (2^16)-1] | 65,536 | unsigned |
| short, short int, signed short, signed short int | %hi | 2 byte | 2 byte | [-(2^15), (2^15)-1] | 65,536 | signed |
| |
| unsigned int | %u | 4 byte | 4 byte | [0, (2^32)-1] | 4,294,967,296 | unsigned |
| int, signed, signed int | %d | 4 byte | 4 byte | [-(2^31), (2^31)-1] | 4,294,967,296 | signed |
| |
| float | %f | 4 byte | 4 byte | [-(2^31), (2^31)-1] | 4,294,967,296 | signed, always |
| double | %lf | 8 byte | 8 byte | [-(2^64), (2^64)-1] | 18,446,744,073,709,551,616 | signed, always |

---

To calculate the range of possible values, use these formulas:
```bash
# Unsigned Range
[0, (2^n)-1]

# Signed Range
[-(2^(n-1)), (2^(n-1))-1] 

n is the number width of the type in bits.
```

These formulas assume 2s complement representation for signed numbers, which is the standard.
